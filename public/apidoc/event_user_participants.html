Event user participants

>>>

POST /api/v1/events/:event_id/event_user_participants
Add staff member to event

Supported Formats

json

Examples

########### Request Example #################
# URL
https://app.divecentrehq.com/api/v1/events/12/event_user_participants

# Request Body
{
  user_token: <USER_TOKEN>,
  store_api_key: <STORE_API_KEY>
  event_id: 12,
  event_user_participant:
    {
      user_id: 1,
      role: "Diver"
    }
}
########### Response Example ################
# SUCCESS
no content

# FAILURE
{
  "event_id": [
    "can't be blank"
  ],
  "user_id": [
    "can't be blank"
  ],
  "role": [
    "can't be blank"
  ]
}

Params

Param Name Description
user_token
required

Value: Must be String
store_api_key
required

Store API Key


Value: Must be String
event_id
required

Event ID


Value: Must be Integer
event_user_participant
required
Value: Must be a Hash
event_user_participant[role]
required

Staff Member role in event


Value: Must be String
event_user_participant[user_id]
required

User ID which you want add to event


Value: Must be Integer
