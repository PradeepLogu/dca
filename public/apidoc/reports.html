PUT /api/v1/reports/financial_reports/:id
Update financial report details

Supported Formats

json

Examples

########### Request Example #################
# URL
https://app.divecentrehq.com/api/v1/reports/financial_reports/1

# Request Body
{
  user_token: <USER_TOKEN>,
  store_api_key: <STORE_API_KEY>,
  store: {
    finance_reports_attributes: {
      "0": {
        finance_report_payments_attributes: {
          "0": {
            custom_amount: 184.00,
            id: 1728
          },
          "1": {
            custom_amount: 508.82,
            id: 1729
          },
          "2": {
            custom_amount: 88.45,
            id: 1730
          }
        },
        id: 890
      }
    },
    tills_attributes: {
      "0": {
        take_out: 0,
        amount: 10.00,
        notes: "Awesome Note",
        id: 491
      }
    }
  }
}

########### Response Example ################
# SUCCESS
no response

#FAILURE
{
  "store": [
    "can't be blank"
  ]
}

Params

Param Name Description
user_token
required

User Token


Value: Must be String
store_api_key
required

Store API Key


Value: Must be String
id
required

Financial Report ID


Value: Must be Integer
store
required
Value: Must be a Hash
store[finance_reports_attributes]
optional , nil allowed
Value: Must be a Hash
store[finance_reports_attributes][finance_report_payments_attributes]
optional , nil allowed
Value: Must be a Hash
store[finance_reports_attributes][finance_report_payments_attributes][custom_amount]
optional

Changed payment amount


Value: Must be Float
store[finance_reports_attributes][finance_report_payments_attributes][id]
optional

Financial report payment id. Should be required if you want update custom_amount value


Value: Must be Integer
store[finance_reports_attributes][id]
optional

Finance Report ID


Value: Must be Integer
store[tills_attributes]
optional , nil allowed
Value: Must be a Hash
store[tills_attributes][take_out]
optional

Till true - then take out, when false - then put in


Value: Must be one of: true, false.
store[tills_attributes][amount]
optional

Amount value


Value: Must be Float
store[tills_attributes][notes]
optional , nil allowed

Till note


Value: Must be String
store[tills_attributes][id]
optional

Till ID


Value: Must be Integer
