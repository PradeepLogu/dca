POST /api/v1/other_events
Create new event

Supported Formats

json

Examples

########### Request Example #################
# URL
https://app.divecentrehq.com/api/v1/events

# Request Body
{
  user_token: <USER_TOKEN>,
  store_api_key: <STORE_API_KEY>,
  other_event: {
    starts_at: "2013-04-30 9:35:40",
    ends_at: "2013-04-30 10:35:40",
    event_type_id: 1, #Selected Dive Trip
    event_trip_id: 12,
    boat_id: 1,
    limit_of_registrations: 10,
    location: "Island",
    number_of_dives: 0,
    frequency: "Daily",
    number_of_frequency: 10
  }
}

########### Response Example ################
# SUCCESS
no content

# FAILURE
{
  "starts_at": [
    "can't be blank",
    "is not a valid datetime"
  ],
  "ends_at": [
    "can't be blank",
    "is not a valid datetime"
  ],
  "event_type": [
    "can't be blank"
  ]
}

Params

Param Name Description
user_token
required

User Token


Value: Must be String
store_api_key
required

Store API Key


Value: Must be String
other_event
required
Value: Must be a Hash
other_event[starts_at]
required

Event start date and time


Value: Must be DateTime
other_event[ends_at]
required

Event end date and time


Value: Must be DateTime
other_event[event_type_id]
required

Event Type ID


Value: Must be Integer
other_event[boat_id]
optional , nil allowed

Boat ID


Value: Must be Integer
other_event[limit_of_registrations]
optional , nil allowed

Limit of registrations for event


Value: Must be Integer
other_event[location]
optional , nil allowed

Event location


Value: Must be String
other_event[number_of_dives]
optional , nil allowed

Number of dives for event


Value: Must be Integer
other_event[frequency]
required

Frequency.


Value: Must be one of: One-off, Daily, Weekly, Fortnightly, Monthly, Yearly.
other_event[number_of_frequency]
optional , nil allowed

Number of frequency. This param is REQUIRED if FREQUENCY NOT EQUAL 'One-off'


Value: Must be Integer
event_trip_id
optional

This param REQUIRED ONLY IF EVENT_TYPE EQUAL “Dive Trip”


Value: Must be Integer
