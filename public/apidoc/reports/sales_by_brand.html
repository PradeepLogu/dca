GET /api/v1/reports/sales_by_brand
Return sum of sales for each brand

Supported Formats

json

Examples

########### Request Example #################
# URL
https://app.divecentrehq.com/api/v1/reports/sales_by_brand

# Request Body
{
  user_token: <USER_TOKEN>,
  store_api_key: <STORE_API_KEY>
  month: 03
  year: 2014
}

########### Response Example ################
{
  "Mares": "$1140.00",
  "Scuba Pro": "$0.00",
  "Cressi": "$0.00",
  "OCEANIC": "$0.00",
  "my brand": "$0.00"
}

Params

Param Name Description
user_token
required

User Token


Value: Must be String
store_api_key
required

Store API Key


Value: Must be String
month
optional

Number of month for which you want receive reports. By default it's current month


Value: Must be Integer
year
optional

Year for which you want receive reports. By default it's current year


Value: Must be Integer
