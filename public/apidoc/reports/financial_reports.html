GET /api/v1/reports/financial_reports
Return list of financial reports for month

Supported Formats

json

Examples

########### Request Example #################
# URL
https://app.divecentrehq.com/api/v1/reports/financial_reports

# Request Body
{
  user_token: <USER_TOKEN>,
  store_api_key: <STORE_API_KEY>
  month: 03
  year: 2014
}

########### Response Example ################
[
  {
    "id": 782,
    "name": "Sales Report (Stores::Invoice): 02 Jun, 2014 10:52am",
    "total_sale": "$207.00"
  },
  {
    "id": 783,
    "name": "Sales Report (Stores::Credit): 02 Jun, 2014 10:52am",
   "total_sale": "$112.00"
  }
]

Params

Param Name Description
user_token
required

User Token


Value: Must be String
store_api_key
required

Store API Key


Value: Must be String
month
optional

Number of month for which you want receive reports. By default it's current month


Value: Must be Integer
year
optional

Year for which you want receive reports. By default it's current year


Value: Must be Integer
