GET /api/v1/events
Show list of all events in the specific store

Supported Formats

json

Examples

########### Request Example #################
# URL
https://app.divecentrehq.com/api/v1/events

# Request Body
{
  user_token: <USER_TOKEN>,
  store_api_key: <STORE_API_KEY>,
  param: "Day"

  #========= OR ==========

  user_token: <USER_TOKEN>,
  store_api_key: <STORE_API_KEY>,
  starts_at: "2014-08-01 09:12:59",
  ends_at: "2014-08-05"
}

Params

Param Name Description
period
optional

Period for events select. By default day


Value: Must be one of: day, week, month, quarter, year.
starts_at
optional

Start at time for event select. If period is empty


Value: Must be DateTime
ends_at
optional

Ends at time for event select. If period is empty


Value: Must be DateTime
user_token
required

User Token


Value: Must be String
store_api_key
required

Store API Key


Value: Must be String
